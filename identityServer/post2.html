Hitting our custom IdentityServer endpoint from Xamarin Forms

Next you will need to hook up your Xamarin Forms App to authenticate against the typical token endpoint used by IdentityServer for a JWT.

Enforcing validation is done the typical way and all the other setup is done the same way as any other IdentityServer4 app. This is the real advantage of using this method for auth. You still have all the other advantages of IdentityServer4.
